Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page as a structured tree of nodes, allowing developers to access, modify, and manipulate the structure, style, and content of HTML and XML documents. Through the DOM, JavaScript can dynamically change the content and layout of a web page.

How do you select an element from the DOM?

You can select an element from the DOM using JavaScript methods like document.getElementById() to select by ID, document.getElementsByClassName() to select by class, and document.querySelector() or document.querySelectorAll() for more flexible CSS-style selectors.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element to manage events for its child elements. Instead of adding multiple event listeners to individual child elements, the parent listens for events that bubble up from the children. This is useful because it improves performance and simplifies event management, especially for dynamically added elements.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using methods like setAttribute() and getAttribute(). To change styles, you can access the style property of an element, e.g., element.style.color = 'red';. You can also add or remove classes using classList, for example, element.classList.add('new-class');.
